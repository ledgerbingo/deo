# DEO Finance Modules

This directory contains all feature modules for DEO Finance, organized in a modular design that is easy for AI agents to understand and use.

## Module Structure

Each module is self-contained and provides:
- Clear interface definitions
- Comprehensive functionality
- Context documentation
- Type safety with TypeScript

## Available Modules

1. **auth** - User authentication and social login
2. **onboarding** - User onboarding, KYC, and service introduction
3. **account** - Wallet and transaction management
4. **card** - Card issuance and management
5. **investment** - Portfolio and investment management
6. **exchange** - Currency exchange (FX) operations
7. **settings** - User profile and application settings
8. **support** - Customer support and help resources
9. **notifications** - System notifications and alerts
10. **dashboard** - Central dashboard aggregating all services

## Design Principles

- **Modularity**: Each feature is isolated in its own module
- **Context**: Each module provides clear context for AI understanding
- **Interface-first**: Well-defined interfaces for all operations
- **Type Safety**: Full TypeScript coverage
- **Minimal Coupling**: Modules communicate through well-defined APIs
