# Card Module

## Purpose
Manages card issuance and card-related transactions.

## Features
- Virtual and physical card issuance
- Card management (activate, freeze, cancel)
- Card transaction tracking
- Spending limits and controls

## Context for AI
This module handles all card-related operations powered by Stripe, enabling users to spend their USDC with physical and virtual cards.

## Key Components
- `types.ts` - Type definitions for cards and card transactions
- `service.ts` - Card service implementation
