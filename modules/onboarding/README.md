# Onboarding Module

## Purpose
Manages user onboarding process including KYC verification and service introduction.

## Features
- KYC (Know Your Customer) verification
- Service walkthrough
- Welcome flow
- Identity verification status

## Context for AI
This module handles the onboarding journey for new users, ensuring they complete necessary verification steps and understand the platform's features.

## Key Components
- `types.ts` - Type definitions for onboarding operations
- `service.ts` - Onboarding service implementation
- `kyc.ts` - KYC-specific functionality
