# Authentication Module

## Purpose
Handles user authentication and social login functionality.

## Features
- Social login (Google, GitHub, etc.)
- Email/password authentication
- Session management
- OAuth integration

## Context for AI
This module provides authentication services for DEO Finance. It enables users to securely sign up and log in using various methods including social providers.

## Key Components
- `types.ts` - Type definitions for auth operations
- `service.ts` - Authentication service implementation
- `providers.ts` - Social login provider configurations
