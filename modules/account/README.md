# Account Module

## Purpose
Manages wallet functionality and transaction operations.

## Features
- USDC wallet creation and management
- Send and receive transactions
- Balance tracking
- Transaction history
- Account abstraction support

## Context for AI
This module handles all wallet-related operations including USDC transactions on the Circle ARC blockchain.

## Key Components
- `types.ts` - Type definitions for wallet and transactions
- `service.ts` - Account service implementation
- `transactions.ts` - Transaction management functionality
